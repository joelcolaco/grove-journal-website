# Grove Journal Website

Static public website for Grove Journal.

Planned production URL: `https://www.joelcolaco.com/GroveJournal`

## What is included

- Plain static HTML/CSS, no build step
- Light Grove Journal visual style
- English, Spanish, French, and German home pages
- Draft Privacy Policy and Terms of Service pages for each language
- GitHub Issue Forms for feature requests and fallback bug reports
- `404.html`, `robots.txt`, `sitemap.xml`, and `.nojekyll`

## GitHub Pages setup

Recommended initial setup:

1. Go to repository Settings > Pages.
2. Set Source to `Deploy from a branch`.
3. Set Branch to `main` and folder to `/ (root)`.
4. Save.

The root `index.html` redirects to `/en/`.
